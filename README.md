## [ДЗ] Shopify — Універсальний банер продукту (Liquid)

## 📋 Опис завдання

Метою було створити універсальний банер продукту, який можна вставляти на сторінку колекції або будь-яку іншу сторінку. Банер має показувати: фото, назву, ціну, мінімум 3 кольори (варіанти з опції Color). При кліку на колір у банері повинна відображатися кількість товару на складі для відповідного варіанту. Якщо запасів немає — показується повідомлення «не доступно». Також потрібно було використати ключові можливості Liquid (assign, capture, for, case/when, render, t тощо), забезпечити fallback-значення та локалізацію.

## ✅ Виконана робота

- Створено секцію banner-product.liquid.
- Реалізовано підключення продукту через параметри:
   - render 'banner-product', product: product
   - render 'banner-product', handle: 'product-handle' (з використанням all_products[...]).
- Додано відображення фото з fallback на placeholder. ALT генерується як: alt → title → дефолт.
- Виведено назву товару з безпечним виводом (escape).
- Додано блок ціни з урахуванням знижки (через capture та грошові фільтри).
- Реалізовано вибір 3 кольорів (визначення індексу опції «Color» з підтримкою різних мовних варіантів).
- Додано обробку кліку по кольору:
   - Якщо inventory_quantity > 0 → показується повідомлення «Доступно: X».
   - Якщо товар відсутній → «Не доступно».
- Винесено блок ціни та кольорові кнопки у підсніпети з підключенням через render (scope).
- Додано бейджі через case/when за брендом (vendor).
- Зроблено кнопку «До кошика» як просте посилання cart?variant=ID&quantity=1.
- Реалізовано міні-галерею зображень для обраного кольору (до 5 фото).
- Тексти "доступно", "недоступно", "оберіть колір" винесено через t з локалізацією українською та англійською.
- Передбачено обробку edge cases: відсутність зображень, ціни або опції «Color».

## 🔧 Технології та підходи

- Shopify Liquid: assign, capture, for, if/else, case/when, uniq, map, where, escape, downcase, default.
- all_products[...] для отримання продукту за handle.
- Робота з варіантами (inventory_quantity, available).
- Грошові фільтри: money, money_without_trailing_zeros.
- Локалізація: t з fallback текстами.
- Розбиття на підсніпети та використання render.
- JS інтеграція для кліку по кольорам та перемикання зображень.

## 📝 Примітки

- Кількість товарів для певного варіанту виведене через alert пералельно з основною логікою перемикання
